输入开头文字，AI 帮你自动续写！
- 支持中英文续写
- 可以调节生成长度和生成几个版本
## 如何在本机运行
1. 克隆仓库  
   `git clone https://github.com/Eliot954/ai-text-genetator-demo.git`
2. 进入文件夹  
   `cd ai-text-genetator-demo`
3. 创建并激活虚拟环境  
   `python -m venv venv` 
   `venv\Scripts\activate`
4. 安装依赖  
   `pip install -r requirements.txt`
5. 启动  
   `streamlit run text_generator_app.py`
打开浏览器访问 http://localhost:8501 即可使用！
